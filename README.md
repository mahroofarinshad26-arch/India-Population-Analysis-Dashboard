# India-Population-Analysis-Dashboard

# 🇮🇳 India Population Analysis Dashboard

##  Project Overview

This project presents an interactive **India Population Analysis Dashboard** created using **Power BI**.

The dashboard provides insights into population distribution based on **age group, gender, and location (rural vs urban)**.

---

##  Objectives

* Analyze total population distribution
* 
* Compare **Male vs Female population**
* 
* Identify **age group trends**
* 
* Study **Rural vs Urban population split**
* 
* Understand **Gender Gap across age groups**

---


---

##  Dashboard Features

###  Page 1: Overview

* KPI Cards (Total Population, Males, Females, Rural)
* 
* Population Trend by Age (Line Chart)
* 
* Rural vs Urban (Pie Chart)
* 
* Population Distribution by Age Group (Bar Chart)
* 
* Male vs Female Comparison (Column Chart)
* 
* Gender Ratio (Donut Chart)
* 
* Age Group Filter (Slicer)

---

###  Page 2: Detailed Analysis

* Population Pyramid (Age vs Gender)
* 
* Gender Gap Analysis (Waterfall Chart)
* 
* Rural vs Urban Distribution (Treemap)
* 
* Population Contribution (100% Bar Chart)
* 
* Data Table (Detailed Values)

---

##  Key Insights

* Majority population belongs to the **Working Age Group**
* 
* **Male population is slightly higher** than female population
* 
* Significant **rural population dominance**
* 
* Gender gap varies across age groups
